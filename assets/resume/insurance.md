| ③ 롯데 운전자        |       ~590만원 |   7% |
| ④ DB 담보         |     ~3,727만원 |  46% |
| **합계**          | **~8,165만원** | 100% |

> ⚠️ **주의**: 갱신형 담보(실손·표적항암 등)는 나이가 들수록 보험료가 오르므로, 실제 평생 납입액은 **1억원 이상**이 될 수 있음. 위 수치는 현재 보험료 고정 기준의 **최소 추정치**

## 🔮 40~50대 보험 로드맵

### 현재 포트폴리오에 없는 것

| 영역               | 현재 보장                | 상태 |
|------------------|----------------------|:--:|
| 실손의료비            | ① 삼성 2세대 실손 90%      | ✅  |
| 암·뇌·심장 진단금       | ④ DB 담보              | ✅  |
| 사망보장             | ① 종신 2천만 + ④ 상해사망 1억 | ⚠️ |
| 운전자              | ③ 롯데                 | ✅  |
| 화재·재난            | ② DB 생활종합            | ✅  |
| **치매·간병**        | **없음**               | ❌  |
| **정기보험 (사망 증액)** | **없음**               | ❌  |

### 40대 — 검토할 보험

#### 1. 치매·간병보험 (우선순위 높음)

- **왜 필요한가**: 치매 간병비는 월 200~400만원, 평균 간병 기간 8~10년. 현재 포트폴리오에 간병 보장이 **전혀 없음**
- **가입 시기**: 40대 초반이 보험료 가장 저렴. 50대 넘으면 보험료 급등하거나 가입 거절
- **추천 담보**: 중증치매진단비, 경증치매진단비, 간병인사용자금(월지급형)
- **예상 월 보험료**: 2~4만원대 (40대 초반, 비갱신 기준)

#### 2. 정기보험 — 사망보장 증액 (가족 부양 시)

- **왜 필요한가**: 현재 사망보장은 종신 2천만 + 상해사망 1억. 질병사망 시 실질 보장은 **2천만원뿐**
- **정기보험이란**: 일정 기간(예: 60세·65세까지)만 보장하는 대신 보험료가 매우 저렴
- **예시**: 사망보장 1억, 60세만기 → 월 2~3만원대
- **포인트**: 자녀 독립 시점(60세 전후)까지만 보장하면 되므로 종신보험보다 정기보험이 효율적

### 50대 — 검토할 보험

#### 3. 유병자보험 (건강 악화 시 대비)

- **왜 필요한가**: 50대부터 건강검진에서 이상 소견이 나오기 시작하면 일반 보험 가입이 어려워짐
- **가입 시기**: 건강할 때 미리 가입하는 게 핵심. 이미 진단받으면 늦음
- **참고**: 현재 ④ DB 담보가 잘 되어있으므로, 건강 유지 시 추가 가입 불필요할 수 있음

#### 4. 연금 (보험보다 연금저축·IRP 우선)

- 보험사 연금보험은 사업비가 높아 **비추**
- **연금저축펀드 + IRP**로 세액공제 받으며 노후 준비하는 게 효율적
- 보험으로 연금을 해결하려 하지 말 것

### 우선순위 정리

| 순위 | 보험           | 시기         | 예상 월 보험료 | 비고                    |
|:--:|--------------|------------|---------:|-----------------------|
| 1  | 치매·간병보험      | **40대 초반** |    2~4만원 | 현재 포트폴리오 최대 공백        |
| 2  | 정기보험 (사망 증액) | 40대        |    2~3만원 | 질병사망 보장이 2천만으로 부족할 경우 |
| 3  | 유병자보험        | 50대 (필요시)  |        - | 건강 상태에 따라 판단          |

> **핵심 원칙**: 새 보험 가입보다 **기존 보험 유지가 우선**. 특히 ① 삼성 2세대 실손과 ④ DB 진단금은 지금 해지하면 다시는 같은 조건으로 가입 불가
